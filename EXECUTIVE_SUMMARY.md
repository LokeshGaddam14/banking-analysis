# Bad Loan Detection System - Executive Summary

## Project Overview
This project implements an automated machine learning system for detecting high-risk loans early, enabling proactive risk management and improving portfolio quality.

## Business Objective
Reduce loan defaults by 35% through early identification of bad loans using predictive analytics and data quality assurance.
